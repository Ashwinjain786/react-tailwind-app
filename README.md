# React + Tailwind App

A modern React app with Tailwind CSS, built with Vite.

## Quick Start

```bash
cd react-tailwind-app
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) and enjoy the beautiful Tailwind-styled counter demo!

## Scripts

- `npm run dev`: Start dev server
- `npm run build`: Build for production
- `npm run preview`: Preview production build
- `npm run lint`: Lint code

Fully responsive, accessible, and production-ready.
